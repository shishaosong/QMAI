## 打包步骤

1. **类型检查**
   ```powershell
   npm run typecheck
   ```
   确保没有 TypeScript 错误

2. **执行打包**
   ```powershell
   npm run build:portable
   ```
   
   这个命令会：
   - 运行 TypeScript 类型检查
   - 构建前端（Vite build）
   - 编译 Rust 后端（Tauri build）
   - 生成可执行文件
