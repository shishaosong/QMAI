## 20260614-18:41 - web-search 指数退避重试（feature/book-analysis-6d-skill）

### ✨ 新增
- ✅ `web-search.ts` 新增 `withRetry` 工具 + `fetchWithRetry` 包装：
  - 5xx / 429 / 网络错误 / 超时 → 指数退避重试
  - 4xx（除 429）→ 不重试，直接返回 null
  - 退避：200ms → 400ms（封顶 2s），加 ≤50ms 抖动
  - 默认最多 3 次尝试
  - 收到 AbortSignal → 立即停止，不重试
- ✅ 注入测试点：`__setWebSearchSleepForTest` 让测试不真等

### 📊 行为变化
- 旧：1 次失败 → 返回 null → LLM 兜底
- 新：1 次失败 → 等 200ms 重试 → 还失败 → 等 400ms 重试 → 还失败 → 返回 null
- 单角色最坏情况耗时：~18s（3×6s timeout + 600ms 等待）；正常抖动恢复无感
